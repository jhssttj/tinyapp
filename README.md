## Dependencies

- Node.js
- Express
- EJS
- bcryptjs
- cookie-session
- mocha
- chai
- nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command or `npm start` to run it using.
